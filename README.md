## Installation ‚ö°

<ul>
<li>
<p>npm</p>
<pre>
<code>npm install npm@latest -g
</code></pre>
    </li>
    <li>
<p>Clone the repo</p>
<pre>
<code>git clone https://github.com/esrasen9/mern-blog-app</code>
</pre>
<li>
<p>Cd into the directories and install npm packages</p>
<pre>
<code> cd api </code>
<code> npm install</code>
<code> cd ../client</code>
<code> npm install</code>

<li>
